Node Health Analyzer es una aplicación de escritorio para la monitorización del estado de las baterías, el mantenimiento predictivo y la detección de anomalías en nodos de adquisición sísmica.

El software importa archivos CSV de estado generados por los nodos Sercel, analiza el rendimiento de las baterías a lo largo del tiempo, detecta comportamientos anómalos, estima la vida útil restante y genera informes operativos.

Características principales:

• Importación masiva de archivos CSV
• Visualización de nodos individuales y múltiples
• Puntuación del estado de la batería
• Clasificación automática de nodos
• Análisis de la tasa de descarga
• Sustitución predictiva de baterías
• Detección de anomalías
• Comparación de nodos en toda la flota
• Informes en PDF y Excel
• Base de datos local SQLite
• Compatibilidad multiplataforma (Windows/Linux)

Desarrollado con Python, PySide6, Pandas, Plotly, SQLite y Scikit-Learn.
